# ESBTP-Yakro – Plateforme éducative

## Contexte
Application web interne utilisée pour la gestion académique
et administrative d’un établissement d’enseignement supérieur.

---

## Fonctionnalités principales
- Gestion des utilisateurs (administrateurs, enseignants, étudiants)
- Gestion académique
- Tableaux de bord
- Sécurisation des accès

---

## Architecture technique
- Backend : Laravel (PHP)
- Base de données : MySQL
- Frontend : Blade / JavaScript
